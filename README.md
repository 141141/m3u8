# M3u8

M3u8 - a mini mu38 downloader written in Golang.

## Features

- Download m3u8 
- Download TS files
- Decrypt TS files
- Merge TS files 

## Usages

**install**

```
go get -u github.com/oopsguy/m3u8
```

**build**

```go
go build
```

**run**

```
./m3u8 -u=http://example.com/index.m3u8 -o=/data/example
```
